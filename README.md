# Machine Learning e Algoritmos Preditivos

Repositório de estudos dedicado à análise de dados, pré-processamento e implementação de modelos de aprendizado de máquina (Machine Learning).

## 🛠️ Tecnologias e Bibliotecas
* **Linguagem:** Python
* **Bibliotecas Principais:** 
    * `Pandas` e `NumPy`: Manipulação e tratamento de dados.
    * `Matplotlib` e `Seaborn`: Visualização de dados e matrizes de correlação.
    * `Scikit-learn`: Implementação de modelos (Regressão, Clusterização, etc.) e métricas.
    * `TensorFlow/Keras`: Redes neurais e Autoencoders.

## 📂 Conteúdos Praticados
* **Exploração de Dados (EDA):** Análise estatística, tratamento de valores ausentes (NaN) e detecção de padrões.
* **Pré-processamento:** Escalonamento de variáveis (MinMaxScaler, StandardScaler) e codificação de categorias (LabelEncoder, get_dummies).
* **Modelos Supervisionados:** Regressão com árvores de decisão (ExtraTreesRegressor).
* **Modelos Não Supervisionados:** Agrupamento com K-Means (técnica de Elbow/Cotovelo) e redução de dimensionalidade (PCA).
* **Deep Learning:** Construção de arquiteturas Autoencoder para reconstrução de dados.

## 🚀 Execução
Os projetos estão contidos em arquivos `.ipynb` (Jupyter Notebook). Para executá-los, recomenda-se o uso do **Google Colab** ou ambiente local com suporte a notebooks:

1. Instale as dependências necessárias (ex: `pip install ucimlrepo tensorflow scikit-learn`).
2. Execute as células sequencialmente para observar as fases de carregamento, tratamento e modelagem.
